# Linting

## Knowledge Check

### What is linting?

Linting means running a tool which will look at your code and enforces a certain style depending on the configuration

### Which problems can linting prevent?

- Minified code having a problem in production
- Scope Conflicts
- Security Problems
- Unreadable code
