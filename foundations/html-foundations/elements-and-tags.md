# Elements and Tags

## Knowledge Check

- What is an HTML tag?
  - An HTML tag is comprised of angle brackets <> with the element inside to indicate the start and end of an HTML element.
  ```
  <> opening tag
  </> closing tag
  ```
  <br />
- What are the three parts of an HTML element?
  1. Opening Tag
  2. Content
  3. Closing Tag
  ```
  <p>content</p>
  ```
